## 🎓 Interview Ace Checklist

### Technical Prep

```
☐ Understand every line of your code
☐ Know embeddings deeply (why 384-dim? tradeoffs?)
☐ Understand cosine similarity (not just formula)
☐ Explain why RAG prevents hallucination
☐ Know why you chose Ollama/Mistral specifically
☐ Have extension ideas ready
☐ Understand trade-offs (accuracy, speed, cost)
```

### Behavioral Prep

```
☐ Why did you build this? (Not: "it's cool", but: thoughtful reasons)
☐ What did you learn? (Specific insights)
☐ What would you do differently? (Show growth mindset)
☐ How would you explain this to [non-technical person]?
☐ Why are you interested in this company?
☐ Questions about their tech that show you've studied
```

### System Design Prep

```
☐ How would you scale this to 1M queries/day?
☐ How would you handle 100M PDFs?
☐ Where would the bottlenecks be?
☐ What database would you use?
☐ How would you ensure quality?
☐ Cost analysis (cloud vs local)
```
